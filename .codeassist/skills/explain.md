# Code Explanation Skill

Explain how code works in a clear, educational manner.

## Steps

1. **Read the code** thoroughly
2. **Identify the purpose** - What problem does it solve?
3. **Trace the flow** - How does data move through it?
4. **Explain key concepts** - Algorithms, patterns, abstractions used
5. **Highlight important details** - Edge cases, gotchas, non-obvious behavior

## Explanation Structure

```
## Overview
One paragraph summary of what this code does.

## How It Works
Step-by-step walkthrough of the logic.

## Key Components
- Component A: Purpose
- Component B: Purpose

## Design Decisions
Why certain approaches were chosen.

## Things to Watch Out For
Non-obvious behavior or gotchas.
```

## Tips

- Use analogies for complex concepts
- Show data flow with examples
- Explain WHY, not just WHAT
- Connect to broader design patterns
